What is Hologram?

Hologram is ruby script that takes a directory of css, sass and/or javascript files and creates a style guide for you from comments found inside your source files.

Why would I use it?

There are other style sheet documentation systems (such as Kneath Style Sheets and styledocco), but they have their own set of limitations or rules that make it difficult to get up and running. To use Hologram all you need to know is markdown, a little bit of yaml, and you can be off and running.

Where can I get it?

Hologram can be found on github, feel free to send pull requests or open issues as well.
