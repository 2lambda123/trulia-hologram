Hologram

What is Hologram?

Hologram is a small ruby script, made by Trulia, that takes an OOCSS project and generates a style guide for you based on comments found within the project. You can find an example project that uses hologram at github. It is similar to Kneath Style Sheets, and a little less similar but related to Styledocco.

Why would I use it?

Hologram was written to make building a style guide as easy as maintaining you css. Documentation is written in your css files using a combination of YAML and Markdown. By using Markdown you can easily format your docs however you wish, and fall back to raw html for those tricky parts that every style guide has.

An important part of any style guide are code examples alongside the visual output. Hologram makes this easy by extending Markdown code blocks that can be marked as examples: 

```html_example
<h1>This is an example that should be displayed on the page and
    as code</h1>
```

```js_example

    as code</h1>
```

Where can I get it?

Hologram can be installed by using gem install hologram
